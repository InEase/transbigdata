TransBigData 为交通时空大数据而生
========================================

TransBigData工具针对时空大数据处理而开发，目前有下面几个功能：

* 添加地图底图
* 火星坐标转换
* 经纬度换算距离
* 栅格渔网划分
* 用KDTree算法匹配点与点、点与线的最近邻
* 出租车GPS数据的OD提取
