***************
快速开始
***************

安装
=============================

在命令提示符中运行下面代码::

  pip install -U transbigdata

在Python中运行下面代码::

  import transbigdata as tbd

依赖包
=============================
TransBigData依赖如下包

* scipy
* pandas
* geopandas
* matplotlib
