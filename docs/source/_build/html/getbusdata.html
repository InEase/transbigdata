数据获取¶

transbigdata.getbusdata(city, keywords)¶

通过输入城市与关键词，获取公交线路的线型与站点

输入

citystr

城市

keywordsList

关键词，线路名称

输出

dataGeoDataFrame

生成的公交线路

stopGeoDataFrame

生成的公交站点

transbigdata.getadmin(keyword, ak, subdistricts=False)¶

输入关键词与高德ak，抓取行政区划gis

输入

keywordsstr

关键词，可以是名称，如”深圳市”，或行政区划编号，如440500

akstr

高德ak

subdistrictsbool

是否输出子行政区划的信息

输出

adminGeoDataFrame

行政区划信息

districtsDataFrame

子行政区划的信息，利用这个可以进一步抓下一级的行政区划
